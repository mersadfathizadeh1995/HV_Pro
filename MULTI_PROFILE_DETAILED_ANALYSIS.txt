### Peak Selection Strategies:

`python
def _select_peak(peaks, freqs, amps, select):
    if select == "leftmost":
        return peaks[np.argmin(freqs[peaks])]
    elif select == "sharpest":
        proms = peak_prominences(amps, peaks)
        return peaks[np.argmax(proms)]
    elif select == "leftmost_sharpest":
        proms = peak_prominences(amps, peaks)
        max_prom = np.max(proms)
        sharp_peaks = peaks[proms >= max_prom * 0.5]
        return sharp_peaks[np.argmin(freqs[sharp_peaks])]
    else:  # "max"
        return peaks[np.argmax(amps[peaks])]
`

### Clarity Ratio Check:

`python
def _check_clarity(freqs, amps, peak_idx, clarity_threshold=1.5):
    """Peak amplitude must exceed clarity_threshold × amplitude at f0/2 and 2*f0"""
    peak_freq = freqs[peak_idx]
    peak_amp = amps[peak_idx]
    
    # Check at f0/2
    if peak_freq/2 >= freqs[0]:
        idx_half = np.argmin(np.abs(freqs - peak_freq/2))
        if peak_amp < clarity_threshold * amps[idx_half]:
            return False
    
    # Check at 2*f0
    if peak_freq*2 <= freqs[-1]:
        idx_double = np.argmin(np.abs(freqs - peak_freq*2))
        if peak_amp < clarity_threshold * amps[idx_double]:
            return False
    
    return True
`

## 6. LEGACY BATCH PROCESSING (hvsr_making_peak.py)

### Architecture:

1. **Parallel Window Processing:**
   - ThreadPoolExecutor with MAX_WORKERS threads
   - Each window independently processed
   - Per-window: FFT, smoothing, QC checks

2. **QC Checks:**
   - STA/LTA ratio (transient detection)
   - Amplitude clipping (saturation)
   - HVSR amplitude bounds (2.0-20.0 default)
   - Flat peak detection (peak/mean ratio)
   - Statistical outlier pass (MAD or IQR)

3. **Peak Detection (Auto Mode):**
   - Uses hvsr_pro.processing.windows.peaks.detect_peaks()
   - Fallback: scipy.signal.find_peaks()
   - Environment variables:
     - HV_AUTO_MODE: 0/1 (enable auto mode)
     - HV_AUTO_NPEAKS: number of peaks (default 3)
     - HV_AUTO_MIN_PROM: prominence threshold (default 0.5)
     - HV_AUTO_MIN_AMP: amplitude threshold (default 2.0)

4. **Output Files:**
   - *.mat: Statistics (VelFreqHV, HVmean, HVStd, HVMedian, HV16, HV84)
   - *_result.json: Per-station data (frequencies, means, percentiles, peaks)
   - Figures: Standard, hvsr_pro, statistics (PNG/PDF)
