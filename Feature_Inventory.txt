### 5.2 AUTO-DETECT PEAK METHODS
**Lines: 116-192** (Primary peak detection)

- **Primary Peak Detection (f0)** (Lines 166-191)
  - **Strategy 1 - Ranged**: If user specifies frequency range, finds max within that band
    * Ignores prominence thresholds
    * Direct search in (fmin, fmax) band
  - **Strategy 2 - Unranged**: Global argmax if no range specified
  - Output: (frequency, amplitude, array_index)
  - Type: Processing Feature

### 5.3 SECONDARY PEAK DETECTION
**Lines: 193-287** (Secondary peaks)

- **Number of Secondary Peaks** (configurable, 0-10)
  - Type: Processing Feature

- **Strategy 1 - Ranged Detection** (Lines 254-270)
  - Finds maximum amplitude within user-specified frequency band
  - Bypasses prominence constraints
  - Skips primary peak and previously claimed secondaries
  - Type: Processing Feature

- **Strategy 2 - Unranged Detection** (Lines 271-279)
  - Uses scipy.signal.find_peaks with prominence threshold
  - Filters by min_amplitude threshold
  - Excludes primary + claimed secondaries
  - Picks strongest candidate
  - Type: Processing Feature

- **Prominence Filter** (configurable)
  - Default: 0.3, Range: 0.0-10.0
  - Scipy find_peaks parameter
  - Type: Processing Feature

- **Amplitude Filter** (configurable)
  - Default: 1.5, Range: 0.0-100.0
  - Minimum peak amplitude threshold
  - Type: Processing Feature

### 5.4 RESULT OUTPUTS
**Lines: 9-26** (ProfileResult class)

- **Per-Profile Data Captured**:
  - name: profile identifier
  - profile: SoilProfile object (Vs structure)
  - freqs: frequency array
  - amps: amplitude array (H/V ratios)
  - f0: primary peak (freq, amp, index) tuple
  - secondary_peaks: list of (freq, amp, index) tuples
  - computed: success flag
  - source_path: input file path
  - Type: Processing Output

### 5.5 SIGNALS EMITTED
**Lines: 35-38**

- progress: (message, current, total) - update messages
- profile_done: (index, ProfileResult) - per-profile completion
- all_done: (list of ProfileResult) - all profiles complete
- error: (error_message) - error handling
