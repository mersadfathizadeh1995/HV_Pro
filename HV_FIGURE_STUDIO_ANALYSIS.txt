### Class
- HVOverlayView (QWidget)

### Displays
- Multi-step HV curves (all steps overlaid)
- Loads from Step* folders in strip_dir
- Each step: hv_curve.csv with freq, amplitude columns
- Color-coded by step with colormap
- Peak markers (*) at max amplitude per step

### Controls
- Log X: Toggle log-scale frequency (default ON)
- Grid: Show/hide grid (default ON)
- Show Peaks: Peak markers (default ON)
- Colormap: 6 options (cividis/viridis/plasma/inferno/tab10/tab20)

### Overlay Method
- _collect_steps(): Scan for Step* folders, load CSV
- Color via cmap(i / max(n-1, 1)) where i = step index
- Plots all steps on single axes
- Peaks marked with np.argmax()

### Save
- None (display-only)

## 5. QWIZARD USAGE - Result: NONE
No QWizard classes found in hvstrip-progressive codebase.

### Custom Wizard Pattern Instead
File: profile_wizard_view.py (Lines 1-599)
Class: ProfileWizardView (QWidget) - NOT QWizard-based

**Manual Step-Through UI:**
- Left: Profile list (QListWidget, 180px)
- Right: HV plot + optional Vs plot
- Bottom: Previous/Next/Finish buttons

**Flow:**
1. set_results(results) → populate list with checkmarks
2. Click profile → _show_profile(idx) → redraw HV + Vs
3. Toggle "Select f0" → _pick_f0 = True (green highlight)
4. Click curve → snap to nearest freq → place marker
5. Repeat for secondary peaks (right-click to remove)
6. Next/Previous → navigate through list
7. Finish → wizard_finished.emit(peak_data)

**No QWizardPage** - Each profile is implicitly a "page"

Class: SaveOptionsDialog (QDialog)

**Structure:**
- Output Directory: QLineEdit + browse button
- Format: QComboBox (PNG/PDF/SVG)
- DPI: QSpinBox (72-600)
- All-Profile Outputs: 9 checkboxes (3 default=TRUE)
  - Combined overlay, HV+Vs, Median HV, Vs comparison, Summary tables
  - Normalized HV, f0 histogram, f0 vs Vs30, Spectral matrix
- Per-Profile Re-save: 3 checkboxes (all default=FALSE)
  - Re-save data CSV, Re-save HV figures, Re-save Vs figures

### Two-Button Save Pattern
1. "💾 Save Results" (Line 492, ui_builder.py)
   - Green (BUTTON_SUCCESS)
   - Calls _save_results()
   - Fast: saves default All-Profile outputs only
   - No dialog

2. "⚙ Save All" (Line 499, ui_builder.py)
   - Gear icon
   - Calls _save_all_results()
   - Opens SaveOptionsDialog
   - Full control: per-profile + advanced outputs
