├── WorkerThread per profile for HV computation
├── Median step (separate navigable phase)
├── Peak picking UI (click on curve)
├── Auto-detect buttons:
│   ├── Auto-detect current
│   ├── Auto All Remaining

    ├── combined_hv_curves.png/pdf
    ├── combined_summary.csv
    ├── median_hv_curve.csv
    └── median_peak_info.txt
`

`
hvsr_making_peak.py (Main Script)
├── Parallel window processing (ThreadPoolExecutor)
├── Per-window computations
│   ├── FFT (Z, N, E)
│   ├── Smoothing (Konno-Ohmachi)
│   ├── QC checks (STA/LTA, clipping, flat peak)
│   └── Rejection tracking
├── Statistics (mean, median, percentiles)
├── Auto-mode (environment-driven)
│   ├── Peak detection (hvsr_pro or scipy)
│   └── Figure generation (3 figure types)

    ├── HVSR_*.mat (statistics)
    ├── HVSR_*_result.json
    └── Figures (PNG/PDF)
`

## KEY DIFFERENCES:

| Feature | Multi-Profile (NEW) | Batch HVSR (OLD) |
|---------|-------------------|-----------------|
| Focus | Multiple profiles, one-by-one | Single station, all windows |
| Workflow | Sequential UI-driven | Batch environment-driven |
| Peak Selection | Interactive + auto | Auto only |
| Median | Explicit step | Computed from windows |
| Threading | Worker per profile | Thread pool for windows |
| Output | Per-profile folders | Single station results |
| Visualization | Interactive wizard | Static figures |
